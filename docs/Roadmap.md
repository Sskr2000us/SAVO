# Roadmap — SAVO

## Phase 0 — Definition
- Finalize target platform (mobile/web)
- Pick 1 device integration for MVP (or manual-only MVP)
- Lock MVP scope and success metrics

## Phase 1 — MVP
- Recipe + step model
- Cooking session engine
- Timer-driven guidance
- Basic session history
- One integration: thermometer OR oven (or none)

## Phase 2 — Reliability
- Better error handling (disconnects, sensor dropouts)
- Calibration / device compatibility matrix
- Telemetry (privacy-respecting)

## Phase 3 — Expansion
- Multi-dish coordination
- Personalization
- More devices
